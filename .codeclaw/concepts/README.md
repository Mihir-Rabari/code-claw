# Concepts

Stable, reusable understanding lives here.

## Rules

- Concepts are distilled from approved proposals and stable observations.
- Keep them short enough to scan.
- A concept should describe a pattern, not an entire history.
- If a concept becomes a commitment, promote it to `decisions/`.

## Suggested filename

`short-slug.md`

## Suggested template

```md
# Concept: <name>

- **Status:** active
- **Derived from:** <proposal or observation links>

## Idea

<the distilled concept>

## Why it matters

<why this helps future work>

## Boundaries

- <what it does not mean>
```
