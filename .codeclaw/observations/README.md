# Observations

Atomic evidence notes live here.

## Rules

- One observation per file.
- Keep observations small, specific, and source-backed.
- Do not turn observations into narratives.
- If something is wrong, add a correction observation rather than rewriting history.

## Suggested filename

`YYYY-MM-DD-short-slug.md`

## Suggested template

```md
# Observation: <short title>

- **Date:** YYYY-MM-DD
- **Source:** <path, issue, command output, PR, or note>
- **Status:** observed

## Observation

<one to three concise bullets>

## Evidence

- <link or citation>

## Notes

- <optional context>
```
