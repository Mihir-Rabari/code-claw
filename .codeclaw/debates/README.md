# Debates

Open questions and unresolved tradeoffs live here.

## Rules

- Use debates when the repo does not yet have a settled answer.
- Record both sides fairly.
- Keep each debate focused on one decision surface.
- Once resolved, move the outcome to `decisions/` and leave a pointer here if useful.

## Suggested filename

`short-slug.md`

## Suggested template

```md
# Debate: <question>

- **Status:** open
- **Related observations:** <links>

## Question

<what is undecided>

## Positions

### Option A
- <pros>
- <cons>

### Option B
- <pros>
- <cons>

## Next step

<what would resolve this>
```
